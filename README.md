# Nisar Ahmed - Health-Tech Architect Portfolio

## Overview
This is the personal portfolio and resume website for Nisar Ahmed, a Health-Tech Product Architect bridging the gap between Healthcare Operations and AI.

## Key Features
- **Design:** Modern, dark-themed aesthetic with "glassmorphism" effects.
- **Tech Stack:** HTML5, CSS3, Vanilla JavaScript.
- **Components:**
  - Interactive "Compliance Console" simulating real-time log monitoring.
  - Responsive navigation and hero section.
  - Operational Experience "War Stories" cards.
